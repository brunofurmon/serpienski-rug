# serpienski-rug
Recursive golang implementation

# Parameters
| Name | Description |
| --- | --- |
| pixelDepth | Image width and height, as powers of 3 |
| factor | Recursion depth |

# Running
Inside your GOPATH folders, run
`go run main.go`
